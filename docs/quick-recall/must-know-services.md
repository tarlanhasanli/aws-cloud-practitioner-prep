# Must-Know Services by Domain (CLF-C02)

## Domain 1: Cloud Concepts
| Service/Concept | One-Line Purpose |
|---|---|
| AWS Global Infrastructure | Regions, Availability Zones, and edge locations provide resilience and global reach. |
| AWS Cloud Adoption Framework (CAF) | Guides business and technical cloud transformation planning. |
| AWS Well-Architected Framework | Design guidance across 6 architecture pillars. |
| AWS Marketplace | Discover and deploy third-party software on AWS. |

## Domain 2: Security and Compliance
| Service | One-Line Purpose |
|---|---|
| AWS Identity and Access Management (IAM) | Control authentication and authorization for AWS resources. |
| AWS Organizations | Manage multiple accounts with centralized governance and billing. |
| AWS CloudTrail | Record API actions for audit and forensic analysis. |
| AWS Config | Track resource configurations and compliance status over time. |
| AWS Artifact | Access AWS compliance reports and agreements. |
| AWS Key Management Service (AWS KMS) | Create and manage encryption keys used by AWS services. |
| AWS Shield | DDoS protection for workloads on AWS. |
| AWS WAF | Filter web requests to block exploits such as SQL injection and XSS. |
| Amazon GuardDuty | Threat detection for suspicious account and workload activity. |
| Amazon Macie | Discover and protect sensitive data in Amazon S3. |
| Amazon Inspector | Find software vulnerabilities and unintended network exposure. |

## Domain 3: Cloud Technology and Services
| Service | One-Line Purpose |
|---|---|
| Amazon EC2 | Resizable virtual machines with OS-level control. |
| AWS Lambda | Serverless event-driven compute without managing servers. |
| Amazon ECS / AWS Fargate | Managed container orchestration with optional serverless runtime. |
| Amazon EKS | Managed Kubernetes control plane. |
| Elastic Load Balancing (ELB) | Distribute traffic across multiple targets. |
| Auto Scaling | Automatically match capacity to demand. |
| Amazon S3 | Durable object storage for broad data workloads. |
| Amazon EBS | Block storage volumes for EC2 instances. |
| Amazon EFS | Shared managed file storage for Linux workloads. |
| Amazon RDS | Managed relational databases with backups and HA options. |
| Amazon DynamoDB | Managed serverless NoSQL key-value/document database. |
| Amazon VPC | Isolated virtual networking for AWS resources. |
| Amazon Route 53 | DNS and routing policy service. |
| Amazon CloudFront | CDN for global low-latency content delivery. |
| Amazon SQS / Amazon SNS | Message queueing and pub/sub event distribution. |
| Amazon EventBridge | Event bus for routing events between AWS and SaaS systems. |

## Domain 4: Billing, Pricing, and Support
| Service/Tool | One-Line Purpose |
|---|---|
| AWS Cost Explorer | Analyze and visualize historical spend and forecasts. |
| AWS Budgets | Set cost/usage budgets and alerts. |
| Cost Anomaly Detection | Detect unusual spend patterns automatically. |
| AWS Pricing Calculator | Estimate architecture cost before deployment. |
| Savings Plans / Reserved Instances | Commitment discounts for predictable usage. |
| Spot Instances | Discounted compute for interruptible workloads. |
| AWS Support Plans | Tiered support offerings from Basic to Enterprise. |
| AWS Trusted Advisor | Recommendations for cost, security, performance, and limits. |
