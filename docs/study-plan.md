# CLF-C02 Study Plan (7 / 14 / 21 Days)

## How to Use This Plan
1. Choose a timeline based on your current AWS familiarity.
2. Study the assigned docs first, then do the related domain questions.
3. Track weak areas and repeat those sections before mock exams.

## Daily Checklist (Use Every Day)
- [ ] Read the assigned section(s)
- [ ] Write 5-10 keyword-to-service flash cues
- [ ] Complete at least 10 practice questions
- [ ] Review wrong answers and write why each distractor is wrong
- [ ] Revisit one comparison set (`RDS vs DynamoDB`, `Amazon CloudWatch vs AWS CloudTrail vs AWS Config`, `AWS WAF vs AWS Shield`)
