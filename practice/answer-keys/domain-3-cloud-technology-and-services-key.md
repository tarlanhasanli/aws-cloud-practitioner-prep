# Domain 3 Answer Key: Cloud Technology and Services

| Q# | Answer | Explanation |
|---|---|---|
| 1 | B | EC2 provides VM instances with OS-level control. |
| 2 | B | Lambda is serverless and event-driven. |
| 3 | B | EKS is AWS managed Kubernetes. |
| 4 | C | EFS is a managed shared NFS file system. |
| 5 | D | Glacier Deep Archive is lowest cost for long-term cold data. |
| 6 | B | Read Replicas scale reads; Multi-AZ focuses on HA. |
| 7 | B | DynamoDB is managed NoSQL for high-scale low-latency access. |
| 8 | B | Amazon VPC is the isolated network boundary. |
| 9 | B | NAT Gateway enables private subnet outbound internet access. |
| 10 | B | Route 53 provides DNS and domain services. |
| 11 | B | CloudFront is the AWS CDN with edge caching. |
| 12 | B | SQS queues messages for asynchronous decoupling. |
| 13 | B | SNS supports publish/subscribe fan-out delivery. |
| 14 | A | EventBridge is an event bus for routing events to targets. |
| 15 | A | Auto Scaling adjusts compute capacity automatically. |
| 16 | B | ALB is Layer 7 and supports host/path routing. |
| 17 | B | NLB handles high-performance Layer 4 traffic. |
| 18 | C | RDS is a managed relational database service. |
| 19 | A | Redshift is designed for analytics/data warehouse workloads. |
| 20 | C | EBS is block storage used for EC2 boot volumes. |
| 21 | C | Glacier tiers are for long-term, infrequently accessed archival data. |
| 22 | A | Storage Gateway provides virtual tape and hybrid storage integration. |
| 23 | B | Direct Connect is dedicated private connectivity to AWS. |
| 24 | B | Site-to-Site VPN uses encrypted tunnels over the internet. |
| 25 | A | CloudFormation is AWS infrastructure as code template service. |
